ninjadog: pug + jinja2 == ❤️

Documentation

Pug template (formerly Jade) support for Python 3.

  • fabric for project automation
  • docopt for the command-line interface
  • leverages the pug-cli library on npm to render pug templates through a subprocess
  • continuous integration through travis-ci
  • tested on multiple version of Python through the use of tox prior to release on pypi
  • Sphinx to build and render documentation prior to deployment on GitHub Pages
